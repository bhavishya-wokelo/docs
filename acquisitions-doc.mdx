---
title: "M&A Activity"
---

Surface M&A transactions across a topic, industry, or market segment. Given a topic (e.g. `"food tech"`, `"cybersecurity"`, `"EV charging infrastructure"`), the API returns a list of completed and announced acquisitions in that space — with structured details on the target, acquirer, deal, and geography.

NOTE: This is an asynchronous API. It returns a `report_id`, which you can use to fetch the [report status](https://docs.wokelo.ai/supporting-apis-doc#report-status) and later call the [download report](https://docs.wokelo.ai/supporting-apis-doc#download-report) endpoint to retrieve the final report.

## Request reference

Here is a sample request containing all parameters:

<CodeGroup>

```bash cURL expandable wrap
curl --location 'https://api.wokelo.ai/api/assets/process_common/' \
--header 'Authorization: Bearer Token' \
--header 'Content-Type: application/json' \
--data '{
  "reportType": "acquisition_workflow",
  "payload": {
    "topic": "Food Tech",
    "workbook_name": "Food Tech",
    "advanced_settings": {
      "company_type": "all",
      "geography": [
        "Worldwide"
      ],
      "funding_stage": [],
      "employee_count": [],
      "total_funding_include_blanks": true
    }
  },
  "reportSource": "WKLAPP"
}'
```

```python Python expandable wrap
import requests
import json

url = "https://api.wokelo.ai/api/assets/process_common/"

payload = json.dumps({
  "reportType": "acquisition_workflow",
  "payload": {
    "topic": "Food Tech",
    "workbook_name": "Food Tech",
    "advanced_settings": {
      "company_type": "all",
      "geography": [
        "Worldwide"
      ],
      "funding_stage": [],
      "employee_count": [],
      "total_funding_include_blanks": True
    }
  },
  "reportSource": "WKLAPP"
})

headers = {
  "Authorization": "Bearer Token",
  "Content-Type": "application/json"
}

response = requests.post(url, headers=headers, data=payload)

print(response.text)
```

</CodeGroup>

The `topic` field specifies the industry, segment, or product category for which M&A transactions should be returned. Use `parameters` to filter the deal universe.

Use `advanced_settings` to filter the acquisition universe by:

| Parameter | Description |
| --- | --- |
| `company_type` | Filter by company type (`private`, `public`, or `all`) |
| `geography` | Geographic scope as [ISO country codes](https://docs.wokelo.ai/iso-country-codes) (e.g. `["USA", "GBR"]`) |
| `funding_stage` | Filter by funding stage |
| `employee_count` | Filter by employee size buckets |
| `total_funding_include_blanks` | Include companies with missing funding data |

Pass empty arrays (`[]`) to include all available options.
