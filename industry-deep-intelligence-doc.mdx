## Quick start

Generates comprehensive industry analysis covering market sizing, trends, transaction activity, partnerships, regulatory environment, and case studies.

`POST  /api/enterprise/industry/enrich/`

NOTE: This is a asynchronous API, which means it will return a request\_id in the response. You would be required to poll the status and then retrieve the results. Read more about async APIs [here](https://docs.wokelo.ai/how-async-apis-work).

Start with a simple API request.

<CodeGroup>

```json cURL expandable wrap
curl --location 'https://api.wokelo.ai/api/enterprise/industry/enrich/' \
--header 'Authorization: Bearer Token' \
--header 'Content-Type: application/json' \
--data '{
    "topic": "Enterprise SaaS security",
    "sections": [
        "market_size"
    ]
}'
```

```json Python expandable wrap
import requests
import json

url = "https://api.wokelo.ai/api/enterprise/industry/enrich/"

payload = json.dumps({
  "topic": "Enterprise SaaS security",
  "sections": [
    "market_size"
  ]
})
headers = {
  'Authorization': 'Bearer Token',
  'Content-Type': 'application/json'
}

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)
```

</CodeGroup>

### Supported sections

| **Section Value** | **What You Get** |
| :-- | :-- |
| market\_size | TAM/SAM/SOM estimates with source citations from tier-1 publishers |
| quant\_insights | Quantitative data: growth rates, adoption metrics, cost benchmarks |
| trends\_and\_innovations | Technology trends, innovation waves, disruption signals |
| transactions\_mna | M&A deal flow: acquirers, targets, deal values, strategic rationale |
| transactions\_fundraising | VC/PE activity: rounds, valuations, key investors, sector heatmap |
| transactions\_ipo | IPO exits: companies, valuations, tickers, listing dates |
| partnerships | Strategic partnership mapping across the industry |
| tier1\_intelligence | Curated insights from McKinsey, BCG, Goldman Sachs, etc. |
| case\_studies | In-depth profiles of 3-4 key companies with strategy timelines |
| industry\_regulations | Regulatory environment: active legislation, compliance requirements |

Geography - list of 3 letter ISO country codes (Eg. \[”USA”, “GBR”\])

Keywords : Focus keywords to narrow analysis (e.g., "zero trust", "SIEM")

Industry Definition: Custom definition of the industry to guide AI analysis.

## Best practices

- **Be specific with industry topic:** Broad terms like "tech" or "finance" will yield generic outputs. Use precise names like "B2B SaaS for supply chain" or "lithium-ion battery manufacturing" to get sharper, more actionable reports.
- **Use parameters.keywords**: add specific keywords that resonate with the industry topic you are researching.
- **Request only needed sections:** Each section adds processing time. Request only what you need for faster results.
