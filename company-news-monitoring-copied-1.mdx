## Overview

Fetch full content and metadata for a specific news article by URL or article ID. Use after `get_company_news` to read a complete article beyond the summary.

## Endpoint Details

- **Method:** GET
- **Endpoint:** `/api/enterprise/company/news/detail/`

## Authentication requirements

- Include a valid JWT token in your request header
- If you don't have a token yet, you can get one from the `/auth/token/` endpoint first.

## Request

### Request Parameters

#### Header Parameters

<ParamField header="Authorization" type="string" required>
  JWT token obtained from the Authentication request
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

## URL Parameters

<ParamField query="url" type="string" required>
  Complete URL of the news article.
</ParamField>

<ParamField query="id" type="string" required>
  News ID. Note: Either of url or id is required.
</ParamField>

## Response

Successful response will return the data in the HTTL response.

```javascript JavaScript - Fetch
const myHeaders = new Headers();
myHeaders.append("Authorization", "Bearer Token");
myHeaders.append("Content-Type", "application/json");

const requestOptions = {
  method: "GET",
  headers: myHeaders,
  redirect: "follow"
};

fetch("https://api.wokelo.ai/api/enterprise/company/news/detail/?url=https://www.defenseworld.net/2026/03/23/nordea-investment-management-ab-lowers-position-in-tesla-inc-tsla.html&id", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.error(error));
```

import (

  method := "GET"

  client := &http.Client {
  }
  req, err := http.NewRequest(method, url, nil)

  if err != nil {
    fmt.Println(err)
    return
  }
  req.Header.Add("Authorization", "Bearer Token")
  req.Header.Add("Content-Type", "application/json")

  res, err := client.Do(req)
  if err != nil {
    fmt.Println(err)
    return
  }
  defer res.Body.Close()

  body, err := io.ReadAll(res.Body)
  if err != nil {
    fmt.Println(err)
    return
  }
  fmt.Println(string(body))
}
