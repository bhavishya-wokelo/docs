title: "Industry Research"
---

End-to-end sector coverage report.

NOTE: This is a asynchronous API, which means it will return a report\_id in the response. You would be required to fetch the [report status](https://docs.wokelo.ai/supporting-apis-doc#report-status) and then use [download report](https://docs.wokelo.ai/supporting-apis-doc#download-report) endpoint to obtain the results.

## Request reference

**Here is a sample request**

```json
curl --location --globoff 'https://api.wokelo.ai/api/workflow_manager/start/' \
--header 'Content-Type: application/json' \
--header 'Authorization: your token' \
--data '{
    "workflow": "industry_primer",
    "industry": "Warehouse Automation"
    
}'
```

Be specific in mentioning the industry name on which the report needs to be prepared.
