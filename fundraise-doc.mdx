---
title: "Fundraising Activity"
---

In-depth fundraising workflow report for a specific company, providing structured intelligence on funding activity, investor landscape, and funding stage analysis.

NOTE: This is an asynchronous API. It returns a `report_id`, which you can use to fetch the [report status](https://docs.wokelo.ai/supporting-apis-doc#report-status) and later call the [download report](https://docs.wokelo.ai/supporting-apis-doc#download-report) endpoint to retrieve the final report.

## Request reference

Here is a sample request containing all parameters:

<CodeGroup>

```bash cURL expandable wrap
curl --location 'https://api.wokelo.ai/api/assets/process_common/' \
--header 'Authorization: Bearer Token' \
--header 'Content-Type: application/json' \
--data '{
  "reportType": "fundraisers_workflow",
  "payload": {
    "topic": "Apple",
    "workbook_name": "Apple",
    "advanced_settings": {
      "company_type": "all",
      "geography": [
        "Worldwide"
      ],
      "funding_stage": [],
      "employee_count": [],
      "total_funding_include_blanks": true
    }
  },
  "reportSource": "WKLAPP"
}'
```

```python Python expandable wrap
import requests
import json

url = "https://api.wokelo.ai/api/assets/process_common/"

payload = json.dumps({
  "reportType": "fundraisers_workflow",
  "payload": {
    "topic": "Apple",
    "workbook_name": "Apple",
    "advanced_settings": {
      "company_type": "all",
      "geography": [
        "Worldwide"
      ],
      "funding_stage": [],
      "employee_count": [],
      "total_funding_include_blanks": True
    }
  },
  "reportSource": "WKLAPP"
})

headers = {
  "Authorization": "Bearer Token",
  "Content-Type": "application/json"
}

response = requests.post(url, headers=headers, data=payload)

print(response.text)
```

</CodeGroup>

The `topic` field specifies the company name for which the fundraising report should be generated.

Use `advanced_settings` to filter the fundraising universe by:

| Parameter | Description |
| --- | --- |
| `company_type` | Filter by company type (`private`, `public`, or `all`) |
| `geography` | Geographic scope for companies |
| `funding_stage` | Filter by funding stage |
| `employee_count` | Filter by employee size buckets |
| `total_funding_include_blanks` | Include companies with missing funding data |

Pass empty arrays (`[]`) to include all available options.
