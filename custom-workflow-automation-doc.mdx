---
title: "Custom Workflow"
---

Build custom research workflows in Wokelo and trigger via API.

NOTE: This is a asynchronous API, which means it will return a report\_id in the response. You would be required to fetch the [report status](https://docs.wokelo.ai/supporting-apis-doc#report-status) and then use [download report](https://docs.wokelo.ai/supporting-apis-doc#download-report) endpoint to obtain the results.

## Request reference

`POST   /api/workflow_manager/start/`

<CodeGroup>

```json cURL expandable wrap
curl --location 'https://api.wokelo.ai/api/workflow_manager/start/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer Token' \
--data '{
    
    "workflow":"industry_research___custom_a842447",
    "industry_research_topic": "Electric vehicles"
    
}'
```

```json Python expandable wrap
import requests
import json

url = "https://api.wokelo.ai/api/workflow_manager/start/"

payload = json.dumps({
  "workflow": "industry_research___custom_a842447",
  "industry_research_topic": "Electric vehicles"
})
headers = {
  'Content-Type': 'application/json',
  'Authorization': 'Bearer Token'
}

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)
```

</CodeGroup>

## Examples

A Sample Request

<CodeGroup>

```json cURL expandable wrap
curl --location 'https://api.wokelo.ai/api/workflow_manager/start/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer Token' \
--data '{
    
    "workflow":"value_creation_e469641",
    "permalink": "air-canada"
    
}'
```

```json Python expandable wrap
import requests
import json

url = "https://api.wokelo.ai/api/workflow_manager/start/"

payload = json.dumps({
  "workflow": "value_creation_e469641",
  "permalink": "air-canada"
})

  'Content-Type': 'application/json',
  'Authorization': 'Bearer Token'
}

print(response.text)
```
